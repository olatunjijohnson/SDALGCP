# Profile Likelihood of Phi

Plots the log-likelihood profile of the spatial scale parameter `phi`.

## Usage

``` r
SDAProfilePhi(obj)
```

## Arguments

- obj:

  an object of class "SDALGCP" from
  [`SDALGCPMCML`](https://olatunjijohnson.github.io/SDALGCP/reference/SDALGCPMCML.md).

## Value

A base R plot.
