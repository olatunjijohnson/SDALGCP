# Population density raster shapefile of Newcastle Upon Tyne in the UK.

A dataset containing the population density of Newcastle upon Tyne of
300 X 300 metres

## Usage

``` r
data(pop_den)
```

## Format

This is a raster file of population density, 300 X 300 meters.

## References

Murdock, A.P., Harfoot, A.J.P., Martin, D., Cockings, S. and Hill, C.
(2015) OpenPopGrid: an open gridded population dataset for England and
Wales. GeoData, University of Southampton.
